is a shortcut or a custom name you assign to a command or a sequence 
creating aliases
	alias ll = "ls -la"
managing alliases 
	alias 
		this command shows all the alliases 
removing alliases 
	unalias ll
creating permanent alias
to make allias permanene add it to ~/.bashrc or ~/.bash_profile
example
alias ll='ls -la'
save and apply 
source ~/.bashrc
