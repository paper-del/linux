/home
	contains personal directories for each user this is where you store your documents application settings and other personal files
/root
	the home directory for the root user separate from the /home directory to ensure the root user can log in even if /home is unavailable
/opt
	reserved for optional or third-party applicaton software packages
/usr
	this directory contains user-installed sofftware and utilities despite its name it generally does not hold individual user's home files. it has its own sub-directory structure such as /usr/bin for non-essential user binaries and /usr/local for software compiled from source
